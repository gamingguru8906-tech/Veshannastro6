Bracelet photo naming guide

Place bracelet photos in this folder using the product SKU as the filename.
The catalog will try these extensions in order: webp, jpg, jpeg, png.

Sequence:
1. VA-BR-TP-001 - Triple Protection Bracelet
2. VA-BR-AM-002 - Amethyst Bracelet
3. VA-BR-CQ-003 - Clear Quartz Bracelet
4. VA-BR-BT-004 - Black Tourmaline Bracelet
5. VA-BR-CT-005 - Citrine Bracelet
6. VA-BR-YA-006 - Yellow Aventurine Bracelet
7. VA-BR-RJ-007 - Red Jasper Bracelet
8. VA-BR-TE-008 - Tiger Eye Bracelet
9. VA-BR-RQ-009 - Rose Quartz Bracelet
10. VA-BR-GA-010 - Green Aventurine Bracelet
11. VA-BR-LL-011 - Lapis Lazuli Bracelet
12. VA-BR-MN-012 - Moonstone Bracelet
13. VA-BR-7C-013 - 7 Chakra Bracelet
14. VA-BR-IO-014 - Iolite Bracelet

Example filenames:
VA-BR-TP-001.webp
VA-BR-AM-002.jpg
VA-BR-CQ-003.png
